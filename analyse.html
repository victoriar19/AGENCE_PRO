Vision globale du projet

Le déploiement d'Internet dans le Haras est un très long projet.
Au semestre 3, je me suis concentrée sur le déploiement de bornes wifis dans le châeau. Ce projet a demandé beaucoup de rigueur pour assurer au client un service de qualité tout en respectant ses exigences d'un point de vu temporel.

Au semestre 4, l'enjeu sera de continuer à développer infrastructure Internet sur les autres bâtiments présents dans le Haras.

Retour sur projet :

Le déploiement du wifi dans le château s'est bien déroulé !
Sur les 29 bornes prévu dans le château seulement une n'a pas été posée car elle est sensée être dans un meuble qui n'est pas encore arrivé.
